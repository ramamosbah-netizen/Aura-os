# Opportunity 360 Command Center

**Date:** 2026-07-12 · **Branch:** `feat/opportunity-360`
**Why:** the opportunity was a pipeline card — value/stage/win% only. The deal is
driven by *qualification*, *who's behind it*, *who we're up against*, and *where it
went*. This turns each opportunity into a command center over the full chain.

## 1. Domain — qualification & win/loss (migration 0153)
`Opportunity` gains **BANT** (`budgetConfirmed`/`authorityConfirmed`/
`needConfirmed`/`timelineConfirmed`, boolean default false), `competitors` and
`source` (text), and `lossReason` (win/loss intelligence). 0153 adds seven
additive columns (booleans NOT NULL default false, rest nullable, guarded @DOWN).
Postgres store COLS/insert/update/map + DTOs + sparse update Pick extended;
create/PATCH coerce the checkbox strings.

## 2. `GET /crm/opportunities/:id/summary`
New `Opportunity360Controller` (literal route before `:id`). Composes the
opportunity, its account, the account stakeholders (with role/strength), the
personal activity timeline, a derived **qualification score** (0–4), the
direct-vs-tender **route**, and the full **progression** it spawned — following
the provenance links the reactors stamp: tenders (`sourceOpportunityId`),
quotations (`sourceOpportunityId` | `sourceTenderId`), contracts (`tenderId`),
projects (`contractId`). Each step reports reached/count/value and links to the
record.

## 3. Opportunity 360 page (`/crm/opportunities/[id]`)
Header (account, Won/Lost/Open pill, route pill, source) with inline **stage**
select and a one-click **→ Quotation** on a won direct sale; snapshot
(value/win%/qualification/close/owner/contracted); **Deal Progression** strip
(opportunity → tender? → quotation → contract → project, clickable); editable
**BANT** checkboxes; **competitors** chips + inline edit; **stakeholders** (from
the account, role/strength, link to Contact 360); **win/loss intelligence**
(contracted value + "beat: …" on win, loss-reason capture on loss).

## 4. Pipeline wiring
Opportunity cards (board + list) link the title to the 360. The create drawer now
captures **source** and **competitors**.
